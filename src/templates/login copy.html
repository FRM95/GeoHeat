How it works?

Obtain fire spots around the globe.
At any time, any date and anywhere you want.

Submit your key and start requesting data directly from FIRMS services. Three.JS library will render and display active fires in a 3D earth representation.

Click at any spot and visualize unique information, filter different fire data by attributes like Date, Source, Area, and Country.

Where is my data going?

A free service without private information.

This platform acts as an intermediate layer between data fetching and data visualization, simplifying the whole process and adding extra information to the original data.

With no user login, you will be able to access the original FIRMS API services while keeping track of your previous calls.
